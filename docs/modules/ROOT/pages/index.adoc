== Installation

If you want to use this extension, you need to add the `io.quarkiverse.quarkus-azure-key-vault:quarkus-azure-key-vault` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.quarkus-azure-key-vault</groupId>
    <artifactId>quarkus-azure-key-vault</artifactId>
</dependency>
----
